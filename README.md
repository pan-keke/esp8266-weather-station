# IoT-Weather-Station-ESP8266

Система мониторинга температуры и влажности в реальном времени на базе NodeMCU и DHT22 с "живым" веб-интерфейсом.

## Особенности
- **Live Updates:** Данные обновляются каждые 2 секунды без перезагрузки страницы (Fetch API).
- **Responsive Design:** Удобный интерфейс в стиле Apple Health, адаптированный под мобильные устройства.
- **High Precision:** Использование датчика DHT22 для точных измерений.

## Схема подключения
| DHT22 Пин | NodeMCU Пин |
|-----------|-------------|
| VCC       | 3V3         |
| Data      | D2 (GPIO4)  |
| GND       | GND         |

## Как использовать
1. Склонируйте репозиторий.
2. Откройте файл `.ino` в Arduino IDE.
3. Установите библиотеки `DHT sensor library` и `Adafruit Unified Sensor`.
4. Введите свои данные Wi-Fi в коде.
5. Прошейте плату NodeMCU.
6. Узнайте IP-адрес через Монитор порта и введите его в браузере.
